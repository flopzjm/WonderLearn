## 🎨 Parámetros de Estilo

### Para DALL-E 3
```
Style: Natural
Quality: Standard or HD
Size: 1024x1024 (cuadrado) o 1024x1792 (vertical)
```

### Para Midjourney
```
--ar 1:1 (cuadrado) o --ar 9:16 (vertical)
--style raw
--stylize 250
--q 2
```

### Prompt Base (agregar al final de cada prompt)
```
kawaii flat design, pastel colors, white/cream background, 
simple shapes, children's book illustration, vector style, 
no text, clean, minimal, educational
```

## 📝 Instrucciones Paso a Paso

### DALL-E 3 (Recomendado)

2. Seleccionar modelo GPT-4 con DALL-E 3
3. Copiar prompt de esta guía

6. Guardar en carpeta correspondiente
7. Renombrar archivo según necesidad

3. Usar comando `/imagine`
4. Pegar prompt + parámetros
5. Esperar generación
6. Descargar imagen
7. Guardar en carpeta correspondiente

## 🎯 Tips de Calidad

1. **Consistencia:** Usar siempre el mismo estilo base
2. **Fondo:** Siempre blanco o crema para fácil integración
3. **Sin texto:** Nunca incluir texto en las imágenes
4. **Resolución:** Usar 1024x1024 como mínimo
5. **Variantes:** Generar 2-3 versiones y elegir la mejor
6. **Nombre de archivo:** Usar convención `nombre_descripcion.png`
